Job Clipper settings

Job Clipper works out of the box: clipped jobs are saved in this browser and tracked in the popup. Connect Trello below if you also want every job on a kanban board you can use anywhere.

1Trello API key

How do I get an API key? Go to trello.com/power-ups/admin → New → name it anything (e.g. "Job Clipper"), pick your workspace → open the API key tab → Generate a new API key → copy it. The key just identifies the app — your token (step 2) is the secret part.
Built in — nothing to do here. ✓

2Trello token

Opens Trello — click Allow, then copy the token it shows you and paste it here.

Creates a "Job Hunt" board with Saved / Applied / Interview / Offer / Rejected lists — or reuses yours if one already exists.

✓ Connected as — syncing to

Your token is stored only in this browser (chrome.storage.local) and is sent only to api.trello.com. It grants read/write access to your whole Trello account (not just the Job Hunt board) and never expires on its own — treat it like a password. You can revoke it any time under Trello → Settings → Applications.
